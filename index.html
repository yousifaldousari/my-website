About

A quick introduction to what I do and how I work.

Kuwait University Mechanical Engineering Capstone - June 2022

Gas Cylinder Cleaning Device

Project overview:
Receiving dirty gas cylinders can be frustrating, to the point where people avoid touching them altogether a common issue in Kuwait. The number of germs and bacteria on those cylinders is concerning, and gas stations often skip cleaning because it is time- and labor-intensive. This project focused on designing a device that cleans gas cylinders quickly and efficiently, making the process more hygienic and practical for both customers and stations.

CODED Academy Capstone Project – July 2025

Smart Seas "AI Marine"

Project overview:
This project assesses marine water quality across Kuwait using data science techniques. By analyzing biological, chemical, and physical parameters collected across seasons, we apply unsupervised machine learning specifically clustering to uncover hidden patterns and classify zones by water condition. This work provides insight into pollution trends and supports proactive marine environmental management.

Skills:

  • Python data analysis
  • K-Means Clustering
  • Data visualization
